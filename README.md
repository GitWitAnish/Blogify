# Blogify

A full-stack blogging platform.

Built with **Node.js**, **Express 5**, **MongoDB**, and **EJS**. Styled with Bootstrap 5. No frontend framework, no build step, no nonsense.

---

## Tech stack

| Layer     | Tech                             |
| --------- | -------------------------------- |
| Runtime   | Node.js                          |
| Framework | Express 5                        |
| Database  | MongoDB + Mongoose               |
| Auth      | JWT (jsonwebtoken) + crypto HMAC |
| Uploads   | Multer (disk storage)            |
| Templates | EJS                              |
| Styling   | Bootstrap 5 + custom CSS         |
| Fonts     | DM Sans, Space Grotesk           |
